Conhecimentos

Html

é a base da web, usando tags para estruturar e exibir conteúdo, como texto, imagens e links, nos navegadores.

Css

dá estilo a páginas web, controlando cores, fontes e layout, para uma apresentação visual atraente.

Javascript

traz vida à web, adicionando interatividade e animações a elementos na página.

Typescript

estende o JavaScript, fornecendo tipagem estática e recursos modernos para desenvolvimento mais robusto.

Angular

simplifica a criação de aplicativos web complexos por meio de uma estrutura organizada.

ReactJS

Enquanto o React é usado para criar interfaces web interativas, o React Native permite desenvolver aplicativos móveis nativos para iOS e Android usando os mesmos princípios do React.

Bootstrap

é um framework front-end que agiliza o design responsivo de sites e aplicativos com estilos predefinidos e componentes.

Git

é um sistema de controle de versão que organiza mudanças no código, simplificando a colaboração e permitindo o gerenciamento do projetos.

Github

é uma plataforma de hospedagem que usa o Git como base, fornecendo ferramentas para compartilhar código, colaborar em projetos e rastrear problemas.
